My Codecademy Challenging Part-2 Project From The Data Scientist Path Foundations of Machine Learning:

Project Goal:
Using a Naive Bayes Classifier Model, classify any tweet (or sentence) and predict whether that sentence came from New York, London, or Paris.

Overview:

In this project, Twitter Classification Cumulative Project, 
I use real tweets to find patterns in the way people use social media. 

There are two parts to this project:
• Part-1: Viral Tweets, Predict Viral Tweets, using a K-Nearest Neighbors classifier model.
• Part-2: Classifying Tweets, Tweet Location (this section)
